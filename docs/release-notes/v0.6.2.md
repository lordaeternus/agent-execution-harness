# v0.6.2

## Fixed

- `claim auto` now batches generated claims in `weak` mode instead of failing when a multi-task plan exceeds the per-action claim limit.
- Verified claims now accumulate across claim batches, so one low-context agent command can still produce a complete final report.

## Validation

- Added a public readiness test covering a 3-task weak-mode plan that generates 12 claims in 2 batches.
- Replayed a temporary no-code simulation and reached `completed` with evidence score `100`.
