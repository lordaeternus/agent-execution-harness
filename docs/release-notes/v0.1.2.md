# Agent Execution Harness v0.1.2

`v0.1.2` is the first share-ready public polish release with the npm/npx installer path verified from a clean directory.

## Highlights

- fixed `npx agent-execution-harness@latest init` template resolution
- clearer README for both non-technical users and developers
- npm/npx quickstart at the top of the docs
- copy-paste prompt for AI coding agents
- troubleshooting for common setup and agent-compliance problems
- stronger `SECURITY.md`
- stronger `CONTRIBUTING.md`
- stricter release-readiness audit
- normalized npm CLI bin metadata
- docs, security policy, and contributing guide included in the npm package
- protected existing `AGENTS.md` files with `--agents-mode skip|append|overwrite`

## Install

```bash
npx agent-execution-harness@latest init --adapter generic --cwd .
```

## Verify

```bash
npx agent-execution-harness@latest doctor --cwd .
```
