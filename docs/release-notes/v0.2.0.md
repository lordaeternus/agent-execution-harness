# Agent Execution Harness v0.2.0

`v0.2.0` adds a token-aware execution layer. The harness now helps agents spend fewer tokens while preserving evidence, claims, auditability and rollback.

Highlights:
- short runtime guide for agents
- compact/full/json report formats
- compact CLI observations
- evidence references with `output_ref` and `sha256`
- full/current artifacts
- macro commands for common execution steps
- conservative auto-claims
- token budget and token benchmark checks

Compatibility:
- Existing `run --action` flows still work.
- Existing run artifacts still load.
- Compact mode is additive and configurable.
